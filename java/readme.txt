Examples for NanoVM for Asuro - http://www.harbaum.org/till/nanovm
------------------------------------------------------------------

These files are to be used with the NanoVM for Asuro and its Convert tool. To
compile e.g. the AsuroLED project just run the suns java compiler using the
command 
  
   javac AsuroLED.java

Asuro specific demo projects:

AsuroLED          - simple LED test. Comes pre-installed with NanoVM for Asuro
AsuroMotor        - a test using the Asuro motors
AsuroLine         - a nice line follower. It follows a black line and turns 180
                    deg if it drives into an obstacle. Start it slightly off 
                    the black line to let it calibrate its sensors.

Generic demos/tests:

HelloWorld        - a simple HelloWorld demo
Inheritance[AB]   - Test for Java inheritance
Erathostenes      - Sieve of Erathostenes
QuickSort         - Implementation of the QuickSort algorithm
Fibonacci         - Recursive Fibonacci test
StringAndHeapTest - Test string and Heap functions (garbage collection)
Switch            - Test the switch statement
